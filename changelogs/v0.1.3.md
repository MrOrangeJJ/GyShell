# v0.1.3 Release Notes

### English

**New Features**
*   **Next-Gen Skill System**: Fully supports the **agentskills.io** specification. Skills can now be folder-based (containing `SKILL.md`) and include supporting scripts or reference materials that the Agent can automatically recognize and use.
*   **SSH Bastion Host Support**: Added support for **Jump Servers (Bastion Hosts)**, allowing secure connections to remote servers in complex network environments.
*   **Interactive Session Auditing**: Introduced an **Action Model** to audit high-risk operations. It automatically validates `send_char` sequences (ensuring correct C0 control character usage) and decides between synchronous/asynchronous execution for `exec_command`.
*   **Rich Input & File Management**: Support for multi-file selection and automatic temporary file management for ultra-long text pastes.

**Optimizations & Refinements**
*   **Stateless Agent Architecture**: Refactored `AgentService` into a stateless execution model, improving session management and system stability.
*   **Enhanced Skill Discovery**: Improved skill scanning logic with support for multiple compatibility directories (including `.claude/skills` and `.agents/skills`).
*   **UI/UX Improvements**: Optimized mention suggestions and error modals using Portals to prevent UI clipping in complex layouts.

### 中文 (Chinese)

**新功能**
*   **下一代技能系统**：全面支持 **agentskills.io** 规范。技能现在支持**文件夹模式**（包含 `SKILL.md`），Agent 可以自动识别并利用技能目录下的配套脚本和参考资料。
*   **SSH 堡垒机支持**：新增对**跳板机 (Bastion Host)** 的支持，满足复杂网络环境下的远程服务器连接需求。
*   **交互审计机制**：引入 **Action Model** 对高风险操作进行审计。自动校验 `send_char` 序列（确保 C0 控制字符的正确使用），并智能决策 `exec_command` 的同步/异步执行模式。
*   **富文本输入与文件管理**：支持多文件选择输入，并针对超长文本粘贴实现了自动化的临时文件管理机制。

**优化与改进**
*   **无状态 Agent 架构**：重构 `AgentService` 为无状态运行模式，提升了会话管理的灵活性和系统整体稳定性。
*   **增强型技能扫描**：优化了技能加载逻辑，支持多个兼容目录（包括 `.claude/skills` 和 `.agents/skills`）。
*   **UI/UX 优化**：使用 Portal 重构了 Mention 建议列表和错误弹窗，解决了在复杂布局下的 UI 遮挡问题。
