# v0.1.5 Release Notes

### English

**Chat History Improvements**

- **Fixed multi-select delete reliability**: deleting selected sessions now executes consistently after confirmation.
- **Removed drag-box selection from Chat History**: selection is now click-based only (single, range, and select-all), with no accidental rectangle selection overlay.
- **Clear progress feedback during delete**: confirmation dialogs now show a loading spinner while delete is running, preventing duplicate clicks and "no response" confusion.

**Export Experience Updates**

- **New export menu in the chat header** with two options:
  - **Export history** (readable Markdown)
  - **Export detailed JSON history**
- **Better export menu positioning**: the popup now opens close to the menu button.
- **More compact export popup UI**: reduced menu size and typography for cleaner density.

**UI Consistency Polish**

- **Unified spacing and hover style** for the three chat header action buttons (`+`, History, More).
- **Queue mode button resized** to a square matching the send button height for better visual balance.

### 中文 (Chinese)

**聊天历史改进**

- **修复多选删除稳定性**：多选后确认删除可稳定执行，不再出现无反应的情况。
- **移除聊天历史拉选框**：历史列表改为纯点击选择（单选、范围选择、全选），不再出现误触发的矩形拉选。
- **删除过程可视化反馈**：删除确认后会显示加载转圈，避免重复点击与“像卡住”的体验。

**导出体验更新**

- **聊天头部新增导出菜单**，支持两种导出：
  - **导出历史记录**（可读 Markdown）
  - **导出详细 JSON 历史记录**
- **导出菜单定位优化**：弹层更贴近按钮，减少视觉跳跃。
- **导出菜单视觉更紧凑**：缩小菜单尺寸与文字，信息密度更合理。

**界面一致性打磨**

- **统一聊天头部三个操作按钮**（`+`、历史、更多）的间距与悬停样式。
- **Queue 模式按钮尺寸调整**：改为与发送按钮同高的正方形，视觉更协调。
